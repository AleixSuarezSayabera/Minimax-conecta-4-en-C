## 4 en Raya en C

Proyecto completo del juego **4 en raya** programado en **C**, con:
- IA usando **Minimax con poda Alpha-Beta**  
- Varios modos de juego 
- Menú completo, niveles de dificultad y configuración del tablero  
- Hay una **IA** implementada con el algoritmo **Minimax**, potenciado con **Alpha-Beta pruning**

---

## Uso

El uso es sencillo, solo hace falta compilar todos los ficcheros C y seguir los menus, donde estan todas las instrucciones.
